# ShopZone - Amazon Style E-commerce Website

A fully functional, responsive e-commerce website built with **React 18**, **Vite**, **Bootstrap 5**, and **React Router**. Features modern Amazon-inspired design with complete shopping cart functionality.

## ✨ Features

- ✅ **Fully Responsive** - Mobile, Tablet, Desktop
- ✅ **React Router** - Home, Products, Product Details, Cart, Checkout, About, Contact
- ✅ **Context API** - Global cart state management
- ✅ **LocalStorage** - Persistent cart data
- ✅ **Search & Filter** - Real-time product search and category filtering
- ✅ **Shopping Cart** - Add/Remove/Update quantity, calculate totals
- ✅ **Product Details** - Full product information with image gallery
- ✅ **Checkout Flow** - Multi-step checkout with form validation
- ✅ **Loading States** - Smooth loading spinners and transitions
- ✅ **Modern UI** - Amazon-inspired design with hover effects and animations
